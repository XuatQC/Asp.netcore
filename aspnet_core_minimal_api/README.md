# aspnet_core_minimal_api

Minimal APIs in ASP.NET Core

# Install Docker

```
https://www.docker.com/products/docker-desktop/
```

# Install DB

```dotnetcli
docker-compose up -d --build
```

# Restore dependencies and project-specific tools

```dotnetcli
cd API
dotnet restore
```

# Run the application

```dotnetcli
cd API
dotnet run
```
